#### 描述

使用vue的服务端渲染技术，开发与上线两套环境

#### 运行方法

``` 
git clone https://github.com/louzhedong/vue-ssr-render

cd vue-ssr-render

npm install

开发环境运行 npm run dev

线上打包 npm run build

线上运行 npm start

或者采用pm2运行 pm2 start process.yml --env production

```

2. axios
